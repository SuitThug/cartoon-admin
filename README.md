# cartoom-admin
漫画云后台管理系统，
这个项目是使用vue3+vite+typeScript+echarts完成的.<br/>
有些功能还有待完善，但是基本的功能都已经实现了.<br/>
有些地方代码我是觉得写得挺烂的，有时间在优化一下.<br/>
完成的功能：用户登陆注册、漫画管理、用户权限管理、菜单动态添加、修改按钮颜色、数据大屏等，<br/>
