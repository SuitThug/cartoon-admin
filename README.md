# cartoom-admin

这个项目是使用vue3+vite+typeScript+echarts完成的，
有些功能还有待完善，但是基本的功能都已经实现了



## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm run dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
